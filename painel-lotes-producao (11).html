Trajetória da produção
Cole o link de convite do grupo. Como navegadores não enviam direto para grupos, o painel abre o WhatsApp Web com a mensagem pronta para você clicar em enviar.
Se preferir enviar para um número específico em vez do grupo. Inclua DDI e DDD.
A cada quantos minutos o painel verifica novos apontamentos.
A sirene toca e uma caixa de alerta aparece nesses horários.
Quando ativo, na virada de turno o WhatsApp Web abre sozinho com o resumo pronto (você só clica enviar). Se desativado, aparece só a caixa de aviso para clicar manualmente.
